Lesson 1: Hello, Web!

In this lesson, you'll create your first web page using HTML.

Step 1: Basic Page Structure

Copy and paste this into your own file:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <p>This is my first website.</p>
  </body>
</html>

Save the file as my-first-page.html and open it in your browser!
